## 0. 観察の目的

このメモは、グラフを見て終わりにせず、次の流れにつなげるために使う。

```text
グラフを見る
↓
観察する
↓
仮説を立てる
↓
特徴量候補にする
↓
モデルで検証する
```

重要:

```text
グラフから分かることは、まず仮説として扱う。
グラフだけで因果関係やモデル性能を断定しない。
```

## 6. 特徴量候補

```text

```

## 9. 観察例

### 例: 時間帯ごとの平均 `cnt`

| 項目 | メモ |
|---|---|
| グラフ種類 | 棒グラフ |
| x軸 | `hr` |
| y軸 | 平均 `cnt` |
| 集計単位 | 時間帯 |
| 集計方法 | 平均 |
| 観察 | 8時、17時、18時の平均`cnt`が高い。3時、4時は低い |
| 解釈 | 通勤時間帯に利用が増えていそう |
| 注意点 | 季節、曜日、天気の影響はまだ分離できていない |
| 仮説 | 平日の通勤時間帯は需要が高い |
| 特徴量候補 | `is_commute_hour`, `workingday_x_commute_hour` |
| 検証方法 | 特徴量追加前後でMAE/RMSE/R2を比較する |

### 例: 月ごとの平均 `cnt`

| 項目 | メモ |
|---|---|
| グラフ種類 | 棒グラフまたは折れ線グラフ |
| x軸 | 年月 |
| y軸 | 平均 `cnt` |
| 集計単位 | 月 |
| 集計方法 | 平均 |
| 観察 | 夏から秋に高く、冬に低い傾向がある |
| 解釈 | 季節性がありそう |
| 注意点 | 折れ線の場合、点と点の間は実測値ではない |
| 仮説 | 季節や月は需要予測に効く |
| 特徴量候補 | `mnth`, `season`, `month_sin`, `month_cos` |
| 検証方法 | 月・季節特徴量の有無で評価指標を比較する |
