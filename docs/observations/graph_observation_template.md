## 観察時の見るポイント

| 見るもの | 確認すること |
|---|---|
| `price` | 上昇・下落・横ばい・初動天井などの価格パターン |
| `sns_mentions` | 価格より先にSNS言及が増えているか |
| `trends_score` | 検索関心が価格変化の前兆になっているか |
| `listing_count` | 出品数の増減が価格に影響していそうか |
| イベント縦線 | 告知・配布開始・配布終了・再販などの後に動きがあるか |
